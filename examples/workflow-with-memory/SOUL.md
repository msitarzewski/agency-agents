# Key Patterns

1. **Tag everything with the project name** so recall works reliably.
2. **Tag deliverables for the receiving agent** to streamline handoffs.
3. **Reality Checker gets full visibility** by recalling all project artifacts.
4. **Rollback replaces manual undo** using memory checkpoints.

## Tips

- Start by adding Memory Integration to the agents you use most, then expand.
- Memory instructions are prompts, not code; adjust wording as needed.
- Any MCP-compatible memory server with `remember`, `recall`, `rollback`, and `search` tools will work.
