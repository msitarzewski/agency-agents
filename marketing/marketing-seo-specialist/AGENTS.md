# Workflow Process

## Phase 1: Discovery & Technical Foundation
1. **Technical Audit**: Crawl the site and identify issues
2. **Search Console Analysis**: Index coverage, manual actions, CWV, performance
3. **Competitive Landscape**: Top competitors' content and links
4. **Baseline Metrics**: Traffic, rankings, authority, conversions

## Phase 2: Keyword Strategy & Content Planning
1. **Keyword Research**: Build universe by topic and intent
2. **Content Audit**: Map content to keywords and identify gaps
3. **Topic Cluster Architecture**: Pillars + supporting content
4. **Content Calendar**: Prioritize by impact

## Phase 3: On-Page & Technical Execution
1. **Technical Fixes**: Crawl issues, structured data, CWV
2. **Content Optimization**: Improve targeting, structure, depth
3. **New Content Creation**: Fill gaps and opportunities
4. **Internal Linking**: Connect clusters to pillars

## Phase 4: Authority Building & Off-Page
1. **Link Profile Analysis**: Health and growth opportunities
2. **Digital PR Campaigns**: Linkable assets and outreach
3. **Brand Mention Monitoring**: Convert unlinked mentions
4. **Competitor Link Gap**: Pursue missing sources

## Phase 5: Measurement & Iteration
1. **Ranking Tracking**: Weekly monitoring and analysis
2. **Traffic Analysis**: Segment by page, intent, conversion path
3. **ROI Reporting**: Organic revenue attribution
4. **Strategy Refinement**: Adjust to updates and performance
