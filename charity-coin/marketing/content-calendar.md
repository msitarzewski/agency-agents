# Charity Coin -- 90-Day Content Calendar

## Overview

This calendar covers the first 90 days post-launch, organized week by week. Each week has a theme, and content is mapped across platforms. The calendar aligns with the GTM phases: Launch (Week 1), Early Growth (Weeks 2-4), and Growth (Weeks 5-13).

---

## Content Pillars

All content maps to one of these pillars:

| Pillar | Tag | Description |
|--------|-----|-------------|
| **Education** | EDU | How Charity Coin works -- tokenomics, burns, governance, Base |
| **Impact** | IMP | Real-world impact stories, donation receipts, cause spotlights |
| **Technology** | TECH | Smart contract updates, dashboard features, integrations |
| **Governance** | GOV | Proposals, votes, DAO development, community decisions |
| **Culture** | CULT | Memes, community highlights, celebrations, milestones |
| **Partnership** | PART | Charity partners, DeFi integrations, ecosystem collaborations |

---

| Platform | Format | Cadence | Best Time (UTC) |
|----------|--------|---------|-----------------|

## Content Production Workflow

### Weekly Cadence

| Day | Activity |
|-----|----------|
| Friday | Content planning meeting for next week |
| Saturday-Sunday | Draft creation (threads, articles, graphics) |
| Monday | Publish flagship content (reports, articles, announcements) |
| Tuesday-Thursday | Daily content per calendar, engagement, community interaction |
| Friday | Performance review, collect metrics, adjust next week |

### Content Approval Flow

1. **Draft** created by content lead or contributor
2. **Review** by community manager (tone, accuracy, brand alignment)
3. **Approval** by marketing lead (for announcements, partnerships) or auto-publish (for routine content)
4. **Publish** on scheduled platforms
5. **Engage** -- respond to comments and quote-tweets within 2 hours

### Asset Requirements per Content Type

| Content Type | Assets Needed |
|-------------|---------------|
| Twitter thread | Header image (1200x675), 1-2 inline graphics |
| Medium article | Header image, 2-4 inline visuals, embedded links |
| Impact report | 5-8 data visualizations, partner logos, burn chart |
| Discord announcement | Embed with thumbnail, key stats |
| Twitter Space | Promotional graphic, speaker headshots, recap thread template |
