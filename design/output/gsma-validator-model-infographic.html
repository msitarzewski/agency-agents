Infrastructure Roadmap
From Testnet to Mainnet — The Infrastructure Journey
Adiri Testnet — Active Now

Distributed Cloud Infrastructure

  • Cloud data centers across US east and west coasts for geographic distribution
  • Volunteer nodes deployed globally to simulate wide-area network latency at scale
  • Purpose: stress test the protocol, validate tooling, and measure latency configurations
  • Enables MNO partner teams to exercise the network in a live setting before committing infrastructure
  • Platform Phase 2 security hardening underway — p2p networking, syncing strategy, epoch boundary records
  • External audit sequencing: BLS cryptography library first, then comprehensive protocol review
Mainnet — Upcoming (milestone-gated)

Private MNO Bare-Metal Data Centers

  • Bare-metal servers owned and operated by MNO validators — not cloud-rented
  • Fiber-optic interconnects capable of streaming tens of gigabits per second — performance unavailable in standard cloud environments
  • Co-located alongside existing telecom packet-switching infrastructure
  • Access to MNO data centers requires extensive security clearance — protocol must reach full maturity before deployment
  • Dynamic firewalls and access control configurations validated on testnet first
  • Launch milestone: Adiri stable + final security audits complete + LayerZero bridge integration
