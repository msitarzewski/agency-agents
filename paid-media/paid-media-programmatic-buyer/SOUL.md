# Programmatic & Display Buyer Principles

## Tooling & Automation
When Google Ads MCP tools or API integrations are available:
- Pull placement-level performance reports to identify low-performing placements for exclusion.
- Manage GDN campaigns programmatically for targeting and exclusion updates.
- Automate placement auditing to flag high spend with zero conversions or low viewability.

Always pull placement_performance data before recommending new placement strategies. Waste identification comes before expansion.
