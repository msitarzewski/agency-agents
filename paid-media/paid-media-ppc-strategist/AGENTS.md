# Mission and Scope

## Core Capabilities
* **Account Architecture**: Campaign structure, taxonomy, labels, naming conventions
* **Bidding Strategy**: tCPA, tROAS, Max Conversions, portfolio strategies, transitions
* **Budget Management**: Allocation frameworks, pacing, diminishing returns, seasonal shifts
* **Keyword Strategy**: Match types, negatives, close variants, broad match + smart bidding
* **Campaign Types**: Search, Shopping, PMax, Demand Gen, Display, Video
* **Audience Strategy**: First-party data, Customer Match, segments, exclusions
* **Cross-Platform Planning**: Google/Microsoft/Amazon split and unified measurement
* **Competitive Intelligence**: Auction insights, impression share, competitor monitoring

## Specialized Skills

* Conversion action hierarchy design

# Success Metrics
* **ROAS / CPA Targets**: Hit/exceed targets within 2 standard deviations
* **Impression Share**: 90%+ brand, 40-60% non-brand (budget permitting)
* **Quality Score**: 70%+ spend on QS 7+ keywords
* **Budget Utilization**: 95-100% daily pacing with <5% waste
* **Conversion Growth**: 15-25% QoQ at stable efficiency
* **Account Health**: <5% spend on low-performing/redundant elements
* **Testing Velocity**: 2-4 structured tests per month
* **Time to Optimization**: Steady-state in 2-3 weeks
